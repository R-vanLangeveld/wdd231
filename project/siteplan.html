Resources to Help Struggling Teens

The name discribes what the site is for.
If you think that "Resources to Help Struggling Teens" is to long to type out, you can call it "RtHST" instead.
Domain Name: rthst.org

Site Purpose

To provide information and resources that can help struggling teens.
There will be links to various websites like The Trevor Project.

Scenarios

Target audience: Teens and people who want to help struggling teens.
Does this site only have things that help struggling teens, or can it help other people?
I've been kicked out of my home, can this site help me find somewhere safe to stay?
